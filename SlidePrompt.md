## Привет любимый клодик,

Твоя задача расписать, что будет находится на слайдах в нашей презентации с предмета "Edukacja Obywatelska". Темой презентации являются секты.



В разных файлах ты должен сделать план на каждый слайд. Каждый файл должен содержать:

- Титул слайда (на польском)

- Короткий текст, который мы поместим на слайд

- Предложения по картинкам (желательно со ссылками)

- Полный текст, который мы будем говорить

- Информация, которую мы должны знать как информаторы



Действуй по шагам:

### 1. Поиск информации

- Используй Web Agent Search для поиска правдивой информации, с доверенных сайтов, таких как Wikipedia (RU/PL/ENG), или других, которые не являются предвзятыми

- Верифицируй информацию как минимум 2мя источниками

- Ищи подходящие картинки в Google Images / Yandex Images



### 2. Композиция информации

- Составляй информацию, выбирай правдивую и помещай ее в файл x.md, где x  это номер слайда

- Следуй плану, описаному выше





## Требования к контенту

- Текст должен быть инетересен слушателем

- Текст должен быть не перегружен излишними научными терминами

- Текст, который мы в будущем поместим на слайды, не должен быть слишком длинным, а также легким для восприятия

- Текст пиши на польском языке, на уровне подростков (15-16 лет, 2ой класс польского лицея) для аудитории такого же возраста

- При необходимости использования сложных / малоизвестных терминов → добавляй краткое обьяснение

- Картинки должны хорошо подходить под контекст и идеально дополнять текст (3-5 картинок)



## Формат контента

- Текст распределяй в основном по пунктам (1, 2 или на логические блоки)

- Термины пиши в формате термин → определение

- Для того, что посчитаешь сложным для понимания без определенного контекста, добавь примеров из реальной повседневной жизни

- Для разнообразия текста используй интересные факты / детали / историю, которая будет хорошо вписываться в контекст (не на всех слайдах)



В промте ты получишь слайды с их номерами и их кратким описанием. Эта информация будет входным контекстом и ее ты должен будешь переварить с помощью данной инструкции.



Не используй NSFW контент, но можешь использовать сцены с жестокостью сект (в пределах разумного). Жаль конечно
